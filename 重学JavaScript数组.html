重学JavaScript数组 🌈

很潦草的一篇JavaScript数组学习笔记

创建数组

一唯数组

通过 new Array() 构造函数创建数组

const arr = new Array("a", "b", "c")

console.log(arr);  // ['a', 'b', 'c']

通过构造函数创建数组，如果只有一个元素不能直接创建，需要用到 Array.of()

Array.of() 和 Array 构造函数之间的区别在于处理整数参数：Array.of(7) 创建一个具有单个元素 7 的数组，而 Array(7) 创建一个长度为7的空数组（注意：这是指一个有7个空位(empty)的数组，而不是由7个undefined组成的数组）。 – 来源 mdn

// const arr = new Array(4)  // 创建了四个空元素
const arr = Array.of(4)  // arr[0] == 4

通过字面量创建数组

consr arr1 = ["a","b","c"]
console.log(arr1);  // ['a', 'b', 'c']

Tips (🎉)

数组是引用类型的数据，通过 typeof 返回数据类型是object(对象)，所以arr改变arrb也会跟着改变。

arr 和 arrb 指向同一块内存地址（🍟）

console.log(typeof (arr));  // object
const arrb = arr
arr[0] = 123456789

console.log(arrb[0]) // 返回123456789
console.log(arrb == arr);  // true

多唯数组创建

通过字面数创建多唯数组

看一眼就知道咋创建的了,访问数组元素的时候通过 [] 来访问

const arr = [[1, 2], [3, 4], [5, 6]]

console.log(arr[0][0]);  // 1
console.log(arr[1][0]);  // 3
console.log(arr[2][0]);  // 5
Tips (🎉)

数组里也可以存放对象

const arr = [{ name: "HTML" }, { name: "CSS" }, { name: "JavaScript" }]

console.log(arr1[0].name);  // HTML
console.log(arr1[1].name);  // CSS
console.log(arr1[2].name);  // JavaScript

数组长度 arr.length

console.log(arr.length);  // 3

也可以通过 arr.length 修改数组长度，arr[3] 并没有值，所以它是一个空的元素。

arr.length = 4 
console.log(arr[3])  // undefined

类型检测与转换

isArray()

console.log(Array.isArray(""));  // 传入字符串返回 false
console.log(*Array*.isArray([]));  // 传入一个数组返回 true

数组转字符串

toString()

const arr = ["a", "b", "c"]

console.log(arr.toString()); // 返回字符串 a,b,c

join()

join()将一个数组 或 类数组通过所有元素连接成一个字符串并返回这个字符串。

console.log(arr.join());  // a,b,c

console.log(arr.join("·"));  // 自定义链接符a·b·c

String()

console.log(String(arr));  //  返回字符串a,b,c

Tips (🍟)

上面这三个方法都不会改变原数组，用到的时候可以 arr = arr.toString(),也可以用一个新的变量来接收它返回值。

字符串转数组

Array.from()

Array.from() 方法从一个类似数组或可迭代对象创建一个新的，浅拷贝的数组实例。 – mdn

const str = "abc"
const arr = Array.from(str)
console.log(arr);  // ['a', 'b', 'c']
Tips (🍟)

使用Array.from() 方法这样来操作Dom就比较方便了

/*
<div id="box">
  <span>1/span>
  <span>2/span>
  <span>3/span>
</div>
*/

let spans = document.getElementById("box").getElementsByTagName("span")

spans = Array.from(spans)

展开语法

…合并数组

const arr1 = [1, 2, 3]
const arr2 = [3, 4, 5]
const arr = [...arr1, ...arr2]

console.log(arr);  // [1, 2, 3, 3, 4, 5]

...args剩余参数

function sun(...args) {
  console.log(args);  // [1, 2, 3, 4, 5]
  return args.reduce((a, b) => a += b)
}

sun(1, 2, 3, 4, 5)

…操作Dom元素

[…spans] 就很灵活

/*
<div id="box">
  <span>1/span>
  <span>2/span>
  <span>3/span>
</div>
*/

let spans = document.getElementById("box").getElementsByTagName("span");
[...spans].map((ele) => {
    ele.addEventListener("click", () => {
        [...spans].forEach(ele => {
            ele.classList.forEach(aclass => {
                if (aclass == "active") {
                    ele.classList.toggle("active")
                }
            })
        })
        ele.classList.toggle("active")
    })
})

… 字符串转数组

展开字符串里的每一个字符为数组的元素。

let str = "abc"
const arr = [...str]  // == ["a","b","c"]

解构语法

  • 解构语法两边的格式要一样。
  • 解构语法可以设置默认值。
  • 变量可以起别名。
解构语法其实用在对象上比较多
const arr = [{ name: "HTML" }, { name: "CSS" }, { name: "JavaScript" }]

let [{ name: html }, { name: css }, { name: javaScript }] = arr  // 别名
console.log(html, css, javaScript); // 分别是HTML CSS JavaScript

let { moe = "猫猫" } = { name: "猫猫" }  // 设置默认值
console.log(moe);  // 猫猫
配合展开语法
const [html, ...args] = ["HTML", "CSS", "JavaScript"]

console.log(html);  // HTML
console.log(args);  // ['CSS', 'JavaScript']

追加元素

通过 [] 添加

因为数组下标是从零开始所以直接使用数组长度[arr.length]来追加元素，手动输入下标追加元素也可以只不过比较麻烦就是了。

let arr = [1, 2]

arr[arr.length] = 3
console.log(arr);  // [1, 2, 3]

push()

push() 方法将一个或多个元素添加到数组的末尾，并返回该数组的新长度(该方法修改原有数组)**。

let arr = [1, 2]

arr.push(3)  // 追加一个元素
arr.push(4,5,6)  // 追加多个元素用逗号分隔
console.log(arr);  //  [1, 2, 3, 4, 5, 6]

也可以配合展开语法追加另一个数组。

arr.push(...[4, 5, 6])

返回值是数组追加元素后的长度。

console.log(arr.push(...[4, 5, 6]));  // 6

unshift()

unshift() 方法将一个或多个元素添加到数组的开头，并返回该数组的新长度(该方法修改原有数组**)**。

let arr = [1, 2]

arr.unshift(1)  // 追加一个元素
arr.unshift(2, 3, 4)  // 追加多个元素用逗号分隔
console.log(arr);  // [2, 3, 4, 1, 1, 2]

移除元素

pop()

pop()方法从数组中删除最后一个元素，并返回该元素的值。此方法更改数组的长度(该方法修改原有数组)**。

对应push()方法，一个是在数组的末尾添加元素，一个是移出数组末尾元素。

let arr = [1, 2, 3, 4, 5, 6]
arr.pop()

console.log(arr);  // [1, 2, 3, 4, 5]

shift()

shift() 方法从数组中删除第一个元素，并返回该元素的值。此方法更改数组的长度(该方法修改原有数组)。

对应unshift()方法，一个是在数组的开头添加元素，一个是移除数组开头的元素。

let arr = [1, 2, 3, 4, 5, 6]
arr.shift()

console.log(arr);  // [2, 3, 4, 5, 6]

截取数组

splice()

传入的前两个参数，分别是开始和结束位置（截取不包括结束位置），后面的参数是截取后在原位置添加的元素。splice()会改变原数组。

let arr = [1, 2, 3, 4, 5, 6]

console.log(arr.splice(0, 2));  // 返回值是被截取的数组元素。 [1, 2]

console.log(arr.splice(0, 1, "a", "b"));  // 截取 [3] ，在被截取的位置添加 'a', 'b'

console.log(arr);  // ['a', 'b', 4, 5, 6]

传入的第一个参数是负数，会从后面开始截取。

let arr = [1, 2, 3, 4, 5, 6]

console.log(arr.splice(-2, 2));  // [5, 6]
console.log(arr);  // 原数组 [1, 2, 3, 4]

如果只传入一个参数，会截取后面所有元素。

let arr = [1, 2, 3, 4, 5, 6]

console.log(arr.splice(2));  // [3, 4, 5, 6]
console.log(arr);  // 原数组 [1, 2]

也可以用splice()来替换某个元素。

arr.splice(1, 1, "猫")

console.log(arr);  // [1, '猫', 3, 4, 5, 6]

移动数组元素。

let arr = [1, 2, 3, 4, 5, 6, 7, 8, 9]

// 移动数组元素

function move(array, from, to) {
  if (from < 0 || to >= array.length) {
    console.error("参数错误")
    return
  }
  const newArror = [...array]
  let item = newArror.splice(from, 1)
  newArror.splice(to, 0, ...item)
  return newArror
}

console.log(move(arr, 2, 4));  //  [1, 2, 4, 5, 3, 6, 7, 8, 9]

slice()

与splice()不同，slice()是提取数组里的元素，slice()提取后是不会改变原数组的。

let arr = [1, 2, 3, 4, 5, 6]

console.log(arr.slice(0, 2));  // [1, 2]

console.log(arr.slice(-2));  // 传入一个负数 [5, 6]
console.log(arr.slice(2));  // 传入一个值 [3, 4, 5, 6]

console.log(arr);  // [1, 2, 3, 4, 5, 6]

清除数组元素

两种常用方法，第一张arr.length = 0，这样arr和arrb都会被清空。

let arr = [1, 2, 3, 4, 5, 6, 7]
let arrb = arr

arr.length = 0

console.log(arrb);  // []

第二种方法是给数组重新赋值为一个空的数组。

let arr = [1, 2, 3, 4, 5, 6, 7]
let arrb = arr

arr = []
console.log(arr);  // []
console.log(arrb);  // [1, 2, 3, 4, 5, 6, 7]

复制数组元素

copyWithin()

语法 arr.copyWithin(target[, start[, end]])

target 必需。复制到指定目标索引位置。
start 可选。元素复制的起始位置。
end 可选。停止复制的索引位置 (默认为 *array*.length)。如果为负值，表示倒数。 
let arr = [1, 2, 3, 4, 5, 6, 7]

console.log(arr.copyWithin(2, 1, 3));  // [1, 2, 2, 3, 5, 6, 7]

查找元素

indexOf() 和lastIndexOf()

indexOf 是从前面开始查找，返回值是数组的下标，找不到返回-1，数组的下标的从0开始的和，lastIndexOf是从后面开始查找。找到了一个就返回下标，就不会继续找了。

const arr = ["HTML", "JavaScript", "CSS", "JavaScript"]

console.log(arr.indexOf("JavaScript"));  // 1
console.log(arr.lastIndexOf("JavaScript"));  // 3

indexOf() 和lastIndexOf()也可以传入第二个参数，指定从哪里开始找。

const arr = ["HTML", "JavaScript", "CSS", "JavaScript"]

console.log(arr.indexOf("JavaScript", 2));  // 3
console.log(arr.lastIndexOf("JavaScript", -2));  // 1

includes()

和上面那俩不同 includes() 返回的是布尔值。

const arr = ["HTML", "JavaScript", "CSS", "JavaScript"]

console.log(arr.includes("JavaScript"));  // true

函数实现includes()

const arr = ["HTML", "JavaScript", "CSS", "JavaScript"]

function includes(arr, find) {
  for (const value of arr) {
    if (value === find) return true
  }
  return false
}

console.log(includes(arr, "JavaScript"));  // true

find()与findIndex()

这两个方法就很es6，传入一个回调函数函数，下面是可选参数。

currentValue 必需。当前元素
index 可选。当前元素的索引值
arr 可选。当前元素所属的数组对象

https://developer.mozilla.org/zh-CN/docs/Web/JavaScript/Reference/Global_Objects/Array/find

const arr = [{ name: "HTML" },{ name: "CSS" },{ name: "JavaScript" },]

// find()
// 返回符合测试条件的第一个数组元素值，如果没有符合条件的则返回 undefined。
const status = arr.find(item => {
  return item.name === "JavaScript"
})

console.log(status);  // {name: 'JavaScript'}

// findIndex()
// 返回符合测试条件的第一个数组元素索引，如果没有符合条件的则返回 -1。
const index = arr.findIndex(item => {
  return item.name === "JavaScript"
})

console.log(index);  // 2

数组排序

sort()

我不理解这个方法的算法，只记住a-b升序，*b-a* 倒序💦

应用场景

给数组里的对象进行排序

手动排序

const arr = [3, 4, 7, 8, 5, 2]

function sort(array) {
  for (const n in array) {
    for (const m in array) {
      if (array[n] > array[m]) {
        const temp = array[m]
        array[m] = array[n]
        array[n] = temp
      }
    }
  }
  return array
}

console.log(sort(arr)); // [8, 7, 5, 4, 3, 2]

迭代方法

forEach()

forEach接收两个参数，第一个产生是一个回调函数，这个函数接受三个参数，分别是每一个元素，元素下标，和数组本身。forEach接受的第二个参数是每次遍历时，前面那个回调函数的this引用。

const arr = [1, 2, 3, 4, 5]

arr.forEach((item, index, arr) => {
  console.log(item, index);
  console.log(arr); //  [1, 2, 3, 4, 5]
})

arr.forEach(function () {
  console.log(this);  // {moe: '猫猫'}
}, { moe: "猫猫" })

map()

这个方法应该是叫映射吧。接收的参数与forEach一样，不同的是forEach是没有返回值的，map是有返回值的，map返回的是一个全新的数组。

const arr = [1, 2, 3, 4, 5]

let res = arr.map((item) => {
  return item += 1
})

console.log(res);  // [2, 3, 4, 5, 6]

filter()

接收的参数与上面一样，怎么描述这个方法呢，当前回调返回的是true当前回调的元素就会被返回。

const arr = [1, 2, 3, 4, 5]

let res = arr.filter((item) => {
  return item <= 3 || item >= 5
})

console.log(res);  // [1, 2, 3, 5]

find() 与 findIndex()

寻找返回true的值，find返回值，findIndex返回下标，与filter不同于查找到第一个返回true的元素find() 与 findIndex()就会停止寻找。

const arr = [1, 2, 3, 4, 5]

let res = arr.find((item) => {
  return item <= 3
})

console.log(res);  // 1

const arr = [1, 2, 3, 4, 5]

let res = arr.findIndex((item) => {
  return item <= 3
})

console.log(res);  // 0

reduce()

如果reduce有第二个参数，会传给前面的回调函数的第一个参数，这第一个参数(回调函数的第一个参数)，会在下一次遍历的时侯，成为上一次遍历的返回值，这里我传的是一个数字，进行了简单的累加6，但是也可以传别的数据类型，比如对象什么的，就很强大。

const arr = [1, 2, 3, 4, 5]

let res = arr.reduce((sum, item) => sum += item, 0)

console.log(res);  // 15

如果reduce没有第二个参数，第一次是第二个值，第二次是上一次的返回值。

const arr = [1, 2, 3, 4, 5]

let res = arr.reduce((a, b) => {
  console.log(a, b);
  // 1 2
  // undefined 3
  // undefined 4
  // undefined 5
  // undefined
})
