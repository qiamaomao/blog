配合展开语法

追加元素

通过 [] 添加

因为数组下标是从零开始所以直接使用数组长度[arr.length]来追加元素，手动输入下标追加元素也可以只不过比较麻烦就是了。

push()

push() 方法将一个或多个元素添加到数组的末尾，并返回该数组的新长度(该方法修改原有数组)**。

也可以配合展开语法追加另一个数组。

返回值是数组追加元素后的长度。

unshift()

unshift() 方法将一个或多个元素添加到数组的开头，并返回该数组的新长度(该方法修改原有数组**)**。

移除元素

pop()

pop()方法从数组中删除最后一个元素，并返回该元素的值。此方法更改数组的长度(该方法修改原有数组)**。

对应push()方法，一个是在数组的末尾添加元素，一个是移出数组末尾元素。

shift()

shift() 方法从数组中删除第一个元素，并返回该元素的值。此方法更改数组的长度(该方法修改原有数组)。

对应unshift()方法，一个是在数组的开头添加元素，一个是移除数组开头的元素。

截取数组

splice()

传入的前两个参数，分别是开始和结束位置（截取不包括结束位置），后面的参数是截取后在原位置添加的元素。splice()会改变原数组。

传入的第一个参数是负数，会从后面开始截取。

如果只传入一个参数，会截取后面所有元素。

也可以用splice()来替换某个元素。

移动数组元素。

slice()

与splice()不同，slice()是提取数组里的元素，slice()提取后是不会改变原数组的。

清除数组元素

两种常用方法，第一张arr.length = 0，这样arr和arrb都会被清空。

第二种方法是给数组重新赋值为一个空的数组。

复制数组元素

copyWithin()

语法 arr.copyWithin(target[, start[, end]])

target 必需。复制到指定目标索引位置。
start 可选。元素复制的起始位置。
end 可选。停止复制的索引位置 (默认为 *array*.length)。如果为负值，表示倒数。

查找元素

indexOf() 和lastIndexOf()

indexOf 是从前面开始查找，返回值是数组的下标，找不到返回-1，数组的下标的从0开始的和，lastIndexOf是从后面开始查找。找到了一个就返回下标，就不会继续找了。

indexOf() 和lastIndexOf()也可以传入第二个参数，指定从哪里开始找。

includes()

和上面那俩不同 includes() 返回的是布尔值。

迭代方法

forEach()

forEach接收两个参数，第一个产生是一个回调函数，这个函数接受三个参数，分别是每一个元素，元素下标，和数组本身。forEach接受的第二个参数是每次遍历时，前面那个回调函数的this引用。

map()

这个方法应该是叫映射吧。接收的参数与forEach一样，不同的是forEach是没有返回值的，map是有返回值的，map返回的是一个全新的数组。

filter()

接收的参数与上面一样，怎么描述这个方法呢，当前回调返回的是true当前回调的元素就会被返回。

find() 与 findIndex()

寻找返回true的值，find返回值，findIndex返回下标，与filter不同于查找到第一个返回true的元素find() 与 findIndex()就会停止寻找。

reduce()

如果reduce有第二个参数，会传给前面的回调函数的第一个参数，这第一个参数(回调函数的第一个参数)，会在下一次遍历的时侯，成为上一次遍历的返回值，这里我传的是一个数字，进行了简单的累加6，但是也可以传别的数据类型，比如对象什么的，就很强大。

如果reduce没有第二个参数，第一次是第二个值，第二次是上一次的返回值。
